\DiaryEntry{Complex Analysis, 3}{2026-03-23}{Complex Analysis}

In this entry we consider mappings between complex functions (in a similar spirit as we considered the mappings induced by the complex exponential in the previous entry).


\subsection{Complex Factor}

The most simple case is multiplication by a complex factor $a$; ie

\bee
w = a \cdot z
\eee

We can see the effect of the mapping more clearly by considering polar coordinates. With $z = r e^{j \theta}$ and  $a = |a| e^{j \alpha}$, we obtain

\bee
w = |a| \cdot z = a r e^{j (\theta + \alpha)}
\eee

Tt follows that the transformation expands or contracts the vector representing $z$ by the factor $|a|$ and rotates it through an angle $\alpha$ about the origin. The image of a given region is therefore geometrically similar to that region. Thus, in particular, a dilation maps a straight line onto a straight line and a circle onto a circle.


\subsection{Linear Mapping}

A bit more complex is the general linear mapping given by

\bee
w = a \cdot z + b
\eee

with $a = |a| e^{j \alpha}$, $b = b_x + j b_y$. This corresponds to a expansion or contraction (easiest represented in polar coordinates) followed by a translation by $b$ (easiest represented in rectangular coordinates).

\subsection{Exponentiation}

\bee
w = z^n
\eee

With $z = r e^{j \theta}$, we obtain $w = r^n e^{j n \theta}$. Below plot shows the effect on lines: Lines on the real and imaginary axis in the $z$-plane are mapped to lines in the $w$-plane; all other lines are mapped to curves.

For example, the curve $x = x, y = t$ gives $z = x + jt \rightarrow w = z^2 = x^2 - t^2 + j 2xt$. This corresponds to $u = x^2 - t^2, v = 2xt$. We can arrange this to $t = v / 2x$ and then we obtain

\bee
u = x^2 - t^2 = x^2 - \frac{v^2}{4x^2}
\eee

which are parabolas.

\subsection{Mapping $1/z$}

This mapping can be rewritten according to

\be\label{2026-03-23:eq1}
w = \frac{1}{z} = \frac{\bar{z}}{|z|^2}
\ee

This is a reflection on the real axis (due to the conjugate operation) and a scaling by $1 / |z|^2$. So points outside the unit circle will be moved inside the unit circle and vice versa. We can also see that by using polar coordinates, we then have

\bee
w = \frac{1}{z} = \frac{1}{re{j \phi}} = \frac{1}{r} e^{- j \phi}
\eee

Lines on the real and imaginary axis are mapped into lines again. This is shown in the plot below.

\begin{figure}[H]
\centering
\includegraphics[scale=0.4]{images/2026-03-23-complex_10.png}
\end{figure}

We start with a line on the real axis, $x = t, y = 0, z = t$. In this case the mapping yields $w = 1/t$. A line on the imaginary axis is given by $z = jt$, the mapping yields $w = -j / t$. \qed


Arbitrary lines are mapped onto circles as shown in the following plot.

\paragraph{Analysis of Mapping of Circles.} To see the effect of the mapping, we start with equation of a circle with radius $r$ and center point $c$,

\bee
|z - c| = r
\eee

Substituing $z = 1/w$, we obtain

\bee
\left| \frac{1}{w} - c \right| = r \rightarrow |\frac{1 - wc}{w}| = r \rightarrow |1 - wc| = r |w|
\eee

We can square both sides; expanding the RHS yields

\bee
(1 - wc)(1 + \overline{wc}) = r^2 |w|^2 \rightarrow 1 - wc - \overline{wc} + |c|^2 |w|^2 = r^2 |w|^2 \rightarrow 1 - wc - \overline{wc} + |w|^2 (|c|^2 - r^2) = 0
\eee

We rearrange terms

\bee
|w|^2 - \frac{wc + \overline{wc}}{|c|^2 - r^2} + \frac{1}{|c|^2 - r^2} = 0 \rightarrow |w|^2 - \frac{wc + \overline{wc}}{|c|^2 - r^2} = - \frac{1}{|c|^2 - r^2} 
\eee

We now complete the square to obain a quadratic expression in $w$,

\bee
\left| w - \frac{\overline{c}}{|c|^2 - r^2}\right|^2 = \frac{|c|^2}{(|c|^2 - r^2)^2} - \frac{1}{|c|^2 - r^2} = \frac{|c|^2 - |c|^2 + r^2}{(|c|^2 - r^2)^2} = \frac{r^2}{(|c|^2 - r^2)^2}
\eee

From this we see that this is a circle with center point 

\be\label{2026-03-23:eq5}
\frac{\overline{c}}{|c|^2 - r^2}
\ee

and radius 

\bee
\frac{r}{|c|^2 - r^2}
\eee

It is woth noting that the center point $c$ maps onto $1 / c = \frac{c_x - jc_y}{c_x^2 + c_y^2}$, but this is \emph{not} not the center of the new circle but the point \eqref{2026-03-23:eq5} instead. This new center point point depends on both $c$ and $r$. It is only the points of the circle in the $z$-plan which are mapped by $1/z$ onto a new circle, and this circle has a different center point. The original center point $c$ is \emph{not} mapped by $1/z$.

Only in the limit of small circles, the center point becomes $\bar{c} / |c|^2 = 1 / c$. \qed

\paragraph{Different Calculation - not finished}

We can make the effect of the mapping more clear by expanding \eqref{2026-03-23:eq1},

\bee
w = u + jv = \frac{1}{z} = \frac{\bar{z}}{|z|^2} = \frac{x - jy}{x^2 + y^2}
\eee

From this we obtain,

\be\label{2026-03-23:eq2}
u = \frac{x}{x^2 + y^2} = \frac{x}{r}, \quad v = \frac{-y}{x^2 + y^2} = - \frac{y}{r}
\ee

with $r = x^2 + y^2$. However, this $r$ is \emph{totally} unrelated to the circle radius $r$ defined below.

We can represent a general circle with center $(c_x, c_y)$ and radius $r$ as

\bee
(x - c_x)^2 + (y - c_y)^2 = r^2
\eee

Expanding this, we obtain

\bee
x^2 - 2x c_x + c_x^2 + y^2 - 2 y c_y + c_y^2 - r^2 = 0 \rightarrow x^2 + y^2 - 2c_x x - 2c_y y + c_x^2 + c_y^2 - r^2 = 0
\eee

When we compare this with the general expression

\bee
A(x^2 + y^2) + Bx + Cy + D = 0
\eee

we see that

\bee
A = 1, \quad B = -2c_x, \quad C = -2c_y, \quad D = c_x^2 + c_y^2 - r^2
\eee



Now we investigate the effect of the complex function: We rewrite \eqref{2026-03-23:eq2} as

\bee
x = ur, \quad y = -vr
\eee

Substituing these back into $r = x^2 + y^2$ yields $r = r^2 (u^2 + v^2) \rightarrow r = \frac{1}{u^2 + v^2}$. Now we substitute back into the original circle equation,

\begin{align*}
A(x^2 + y^2) + Bx + Cy + D &= A r + B ur - C vr  + D \\ 
&= A \frac{1}{u^2 + v^2} + B \frac{u}{u^2 + v^2} +C \frac{-v}{u^2 + v^2} + D = 0    
\end{align*}

Multiplying with $u^2 + v^2$ yields

\bee
D (u^2 + v^2) + B u - C v + A = 0
\eee

and dividing by $D$ we finally obtain

\bee
u^2 + v^2 + \frac{B}{D} u - \frac{C}{D} v + \frac{A}{D} = 0
\eee

and this is again a circle equation. But I'm not sure how we can obtain the cente rpoint and radius of this circle in the $u-v-$plane in terms of $c$ and $r$ like we did in the previous subsection.
